Rank Exploits

Instead of striving for a suitable method to obtain a single vector representation of a large document of text, we aim to develop a similarity metric that makes use of the sim- ilarities between the individual embedded word vectors in a document and a query

Methodology

  • Scrape
    • Crawling website and storing data
    • Sanitizing data (stemming, punctuation)
  • Indexing
    • Crawling website and storing data
    • Sanitizing data (stemming, punctuation)
  • Ranking
    • Use K-nearest neighbors to extract the closest documents to the input query
    • Calculate similarity (cosine, euclidean)
    • Use tf-idf to rank documents based on the input query
  • Evaluation
